# Portfolio CMS

A production-style Content Management System (CMS) built on AWS to power my personal portfolio website.

**Public Website**

https://jadonmaldonado.com

This project is part of my Cloud Engineer portfolio and is being designed using real-world engineering practices with an emphasis on AWS architecture, networking, security, monitoring, and operational excellence.

---

## Project Goals

This project demonstrates experience with:

- AWS VPC Networking
- EC2
- Application Load Balancer
- Auto Scaling
- Amazon RDS
- Amazon S3
- Route 53
- AWS Certificate Manager (ACM)
- IAM
- CloudWatch
- High Availability
- Secure Application Architecture

---

## Planned Features

- Secure administrator login
- Manage portfolio projects
- Manage certifications
- Update About Me section
- Upload resume
- Upload project screenshots
- Publish content to the public website
- Contact form management
